# 003 - Atelier NightClazz: Introduction à Vue.js

Atelier NightClazz: Introduction à Vue.js

Aurelien Loyer
  Slides

À la fin de cette NightClazz, vous serez dans la capacité de développer une application Vue.js simple comprenant les éléments suivants:
- Composantes
- Directives
- Requêtes HTTP
- Routeur
- Formulaire
- VueX.

Une brève présentation théorique sur Vue.js sera suivie de l'atelier hands-on où vous pourrez bâtir votre première application Vue.js.
